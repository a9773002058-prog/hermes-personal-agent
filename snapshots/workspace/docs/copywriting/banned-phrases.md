# Banned Phrases

Avoid:

- "100% результат"
- "гарантированно"
- "навсегда избавитесь"
- "чудо-метод"
- "секрет, который скрывают врачи"
- "срочно, пока не поздно"
- "вы обязаны"
- "иначе будет плохо"
- "единственный способ"
- "лечит все"

Replace pressure with calm expert explanation and a soft next step.
